# 🪵 Skeuomorphism Portfolio - Gautham Krishna B

A rich, realistic portfolio design featuring textures, depth, and tactile elements that mimic physical materials like leather, wood, and gold.

## 🎨 Design Philosophy

Skeuomorphism creates interfaces that mimic real-world materials and textures. This portfolio uses:
- **Realistic gradients** mimicking aged paper and leather
- **Multi-layered shadows** for authentic depth
- **Embossed/debossed effects** for tactile feedback
- **Gold accents** for premium feel
- **Beveled edges** on interactive elements
